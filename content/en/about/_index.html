Why use a visual language based on finite state automata?

{{< /blocks/cover >}} {{% blocks/lead %}} The goal of this project is to determine whether a visual programming language using finite state automata achieves better results compared to a interlocking block language like blockly. To test this I developed Automaduino. You can find our study results here soon. {{% /blocks/lead %}}
